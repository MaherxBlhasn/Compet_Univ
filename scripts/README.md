# Dossier Scripts

Ce dossier contient les scripts utilitaires et d'optimisation.

## Fichiers

### Scripts d'optimisation
- **optimize_example.py** : Script principal d'optimisation des surveillances avec CP-SAT

### Scripts de génération
- **generate_jour_seance.py** : Génération des mappings jours/séances

### Scripts de diagnostic et vérification
- **check_quotas.py** : Vérification des quotas des enseignants
- **check_tables.py** : Vérification de la structure des tables
- **diagnostic.py** : Diagnostic général du système
- **infeasibility_diagnostic.py** : Analyse des problèmes d'infaisabilité
- **surveillance_stats.py** : Statistiques sur les surveillances

### Modules
- **quota_enseignant_module.py** : Module de gestion des quotas des enseignants
